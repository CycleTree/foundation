# 2026-03-01 活動記録

## iCall v0.1 開発

### やったこと
- Twilio + OpenClaw を連携させた AI 電話システムを一から構築
- OpenAI Realtime API で日本語の音声認識（STT）を実装
- OpenAI TTS (Nova ボイス) で音声合成を実装
- anagi (Claude Opus) と電話で会話できるようになった
- 実際に電話で「今日の天気は？」「あなたの動いてるOSは？」などの会話をテスト
- ls コマンドの実行結果を音声で返すところまで確認

### できなかったこと
- systemd サービス化（時間切れ）
- 着信対応（発信のみ実装）

### 苦労したこと
- **音声がノイズだらけ問題**: OpenAI TTS → mu-law 8kHz 変換で音が壊れた。原因は自前の mu-law エンコーダがバグってた。`alawmulaw` パッケージに置き換えて解決。
- **Claude Opus が遅すぎ問題**: 応答に15秒かかって、ユーザーが待てずに切ってしまう。「はい、少々お待ちください」を先に流す workaround で対処。
- **サーバーが勝手に落ちる**: exec の background process がすぐ死ぬ。`setsid` + `disown` で解決。

### 学び
- Twilio Media Streams は mu-law 8kHz mono 固定
- OpenAI Realtime API は `gpt-4o-realtime-preview` を使う
